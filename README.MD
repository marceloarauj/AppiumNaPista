# Instalando pacotes necessários para rodar o projeto (cmd)

- npm install appium
- npm install webdriverio
- npm install appium-uiautomator2-driver

# Instalando os packages adicionados no package.json

- npm install

# Executando o projeto (cmd)

- node index.js

# link para comandos do driver 

- http://appium.io/docs/en/about-appium/api/

# link para comandos do element

- https://webdriver.io/docs/api.html

# obtendo elementos da tela

Para facilitar a visualização dos elementos, será utilizado o uiautomatorviewer.bat
que está localizado em AppData\Local\Android\Sdk\tools\bin
para isso será necessário instalar o sdk tools pelo android studio
Ao iniciar, selecione a opção Device Screenshot (uiautomator dump) para visualizar a árvore
de elementos na tela.
